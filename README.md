# 🚀 LaunchPad – Student Startup Accelerator

A responsive landing page built with pure HTML & CSS.

## 📌 Task Objective
Build a clean, responsive landing page with a header,
hero section, features, and footer.

## 🛠️ Tools Used
- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- VS Code + Live Server Extension
- Google Fonts (Syne + DM Sans)
- Font Awesome Icons

## 🌟 Features
- Fixed responsive header with hamburger menu
- Animated hero section with floating orbs
- Stats bar, feature cards, how-it-works steps
- Mentor section, CTA email form
- Fully mobile responsive (640px & 900px breakpoints)

## 🚀 How to Run
1. Clone this repo or download ZIP
2. Open `index.html` in any browser
   OR use VS Code Live Server extension
